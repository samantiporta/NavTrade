# NavTrade
A full-stack trading journal for logging trades, tracking performance, and surfacing patterns in your trading behavior.

## Features
-> Log trades (ticker, entry/exit price, size, date, direction, notes)
-> Full CRUD — create, view, edit, and delete trades
-> User authentication (each user's trades are private)
-> Stats dashboard — win rate, average win/loss, total P&L, best/worst trade
-> Filter and sort trades by date, ticker, or outcome

## Tech Stack
-> Frontend: React (JavaScript)
-> Backend: FastAPI (Python)
-> Database: PostgreSQL
-> ORM: SQLAlchemy
-> Validation: Pydantic
-> Deployment: [e.g. Vercel (frontend) + Railway/Render (backend + DB)]
